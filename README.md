# gmall
商城系统

gmall-core核心模块

1. 包含了通用依赖：
   nacos注册中心、配置中心
   mybatis-plus
   mysql驱动
   swagger
   lombok
2. 通用工具类：
   请求参数包装类
   通用的响应数据
   日期工具类
   sql过滤工具类
3. 自定义异常



springboot的常用依赖有：

- data-redis
- web
- security

springCloud的常用依赖有：

- hystrix
- openfeign
- alibaba-nacos-discovery（已经在core中引入）
- alibaba-nacos-config（已经在core中引入）
